# Contributing to Arun's Portfolio

Thank you for your interest in contributing to this portfolio project!

## Development Setup

1. Fork and clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Build for production: `npm run build`

## Code Standards

- Follow existing code style (Prettier formatting)
- Write meaningful commit messages
- Test all changes locally before submitting
- Ensure accessibility features are maintained

## Pull Request Process

1. Create a feature branch from `main`
2. Make your changes
3. Run `npm run build` to ensure no errors
4. Submit a pull request with a clear description

## Areas for Contribution

- Bug fixes
- Accessibility improvements
- Performance optimizations
- Design enhancements
- Content updates

## Questions?

Open an issue for any questions or suggestions.
